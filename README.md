# gemini-tab-title

A minimal browser extension that sets the tab title to the current [Gemini](https://gemini.google.com) conversation name — mirroring the behavior ChatGPT has natively.

Compatible with Firefox and all Chromium-based browsers (Chrome, Edge, Brave, etc.).

## Installation

**Chromium:** `chrome://extensions` → Enable Developer mode → Load unpacked → select this folder

**Firefox:** `about:debugging#/runtime/this-firefox` → Load Temporary Add-on → select `manifest.json`
